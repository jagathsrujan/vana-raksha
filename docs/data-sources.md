# VanaRaksha — Data Sources Reference

## Satellite & Remote Sensing
| Source | Data | Access |
|---|---|---|
| Google Earth Engine | NDVI, LST, impervious surface | GEE API (free) |
| MODIS MOD13Q1 | 250m vegetation indices | GEE or direct download |
| MODIS MOD11A2 | 1km Land Surface Temperature | GEE |
| Sentinel-2 L2A | 10m land cover, NDBI | GEE, Copernicus API |
| Copernicus GLO-30 | 30m global land cover | Direct download |
| SRTM DEM | 30m digital elevation | GEE or USGS |
